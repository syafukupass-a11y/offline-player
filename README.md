# Offline Player

端末内の動画をブラウザに保存し、通信なしで再生できるPWAです。動画は外部サーバーへ送信されません。

## 主な機能

- 複数動画の一括取り込みと端末内保存（IndexedDB）
- 複数プレイリストの作成、追加、削除、ドラッグ並べ替え
- 連続再生、前後移動、シャッフル、1本リピート
- お気に入り、検索、並び替え、再生位置の記憶
- ホーム画面へのインストールとオフライン起動

## 起動方法

PWAと端末保存機能のため、`file://` で直接開かずHTTPサーバーで配信します。

```bash
python3 -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。同じWi-Fi内のスマートフォンから試す場合は、PCのIPアドレスを使って `http://PCのIP:8080` を開きます（PWAのインストールにはHTTPSが必要な場合があります）。

## iPhone / iPad

Safariで公開URLを開き、共有ボタンから「ホーム画面に追加」を選びます。iOSは空き容量に応じてWebサイトデータを整理する場合があるため、大切な元動画は別途保管してください。

## 注意

- 保存可能容量はブラウザと端末の空き容量に依存します。
- 対応形式は端末ブラウザが再生できる形式に依存します。互換性重視ならMP4（H.264/AAC）がおすすめです。
- 他サイトの動画をダウンロードする機能はありません。権利を持つ動画・端末内の動画を利用してください。
